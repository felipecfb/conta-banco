## Desafio Proposto

Importar a classe Scanner e criar uma conta bancária para o usuário pelo terminal. Após isso, deve exibir as informações dessa conta bancária, por exemplo:

"Olá [Nome do Cliente]..."

## Dados

_Número da conta - Int_
_Agência - Texto_
_Nome do Cliente - Texto_
_Saldo - Decimal_
